Web Download

Download from HuggingFace. Fast, global CDN but you need to get access to the model repository and provide your user token. 

wget --header="'Authorization: Bearer <user_token>'" https://huggingface.co/CompVis/stable-diffusion-v-1-4-original/resolve/main/sd-v1-4.ckpt -O sd-v1-4.ckpt

Alternatively, you could use this Google Drive link that the author of the WebUI shared:
